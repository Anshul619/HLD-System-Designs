# Document DB

The following table compares terminology used by document databases with terminology used by relational databases.

| Relational          | Document          |
|---------------------|-------------------|
| Table               | Collection        |
| Row                 | Document          |
| Column              | Field             |
| Primary key         | Object ID         |
| Nested table/object | Embedded document |
