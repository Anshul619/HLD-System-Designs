# Git

| Command                                         | Remarks                                                                                                                                |
|-------------------------------------------------|----------------------------------------------------------------------------------------------------------------------------------------|
| git bisect                                      | Use git bisect to compare the buggy commit to an early commit that works as expected.                                                  |
| --global                                        | Apply git configurations across your entire git environment                                                                            |
| Rebasing                                        | With the rebase command, you can take all the changes that were committed on one branch and replay them on a different branch.         |
| git-diff-tree                                   | Compares the content and mode of blobs found via two tree objects.                                                                     |
| Staging area                                    | Files stored before they are committed to the local repository                                                                         |
| git show refs/head/push-notifications           | References to `push-notifications` branch                                                                                              |
| --amend                                         | Modify the commit                                                                                                                      |
| git stash                                       | Use git stash to save your work and come back later and reapply the stashed commit.                                                    |
| git octopus                                     | git octopus allows you to merge all you feature branches together at any moment.                                                       |
| git reset myFile.txt                            | After mistakenly staging a file named myFile to the index, how would you remove it from the index to exclude it from your next commit? |
| git merge --abort                               | Stop the merge and restore to the pre-merge state.                                                                                     |
| git stash pop                                   | git stash pop places a saved state from the stash list onto the working directory.                                                     |
| git clean -f                                    | Remove the untracked files from the working directory                                                                                  |
