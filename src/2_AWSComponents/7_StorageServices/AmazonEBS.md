# Amazon Elastic Block Store (EBS)
- [Amazon Elastic Block Store (Amazon EBS)](https://aws.amazon.com/ebs/) is an easy-to-use, scalable, high-performance block-storage service designed for [Amazon Elastic Compute Cloud (Amazon EC2)](../4_ComputeServices/AmazonEC2/ReadMe.md).
